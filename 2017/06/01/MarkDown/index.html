前言

说到md文件可能很多人都曾在自己电脑上看过，但是又不明确md文件究竟是干嘛用的，简单来说Markdown 是一种轻量级的「标记语言」，它的优点很多，目前也被越来越多的写作爱好者，撰稿者广泛使用。Markdown 的语法十分简单，对于初学者来说学习成本也比较低，所以推荐大家对其进行了解。
具体详细说明可参考 Markdown的百度百科

书写工具

Markdown的书写工具众多，在各个操作系统都有客户端或网页端能够进行书写，因此不过多叙述，请自行查找。

简单语法规则

标题

如果想要让一段文字变为标题，在其前面加上 # 即可，推荐 # 与标题之间留有间隙。Markdown的标题共有六级，一级最大，六级最小，# 的个数决定了标题的大小级别。(需要注意的是标题与其它文字会有较大的上下边距)

列表

无序列表：在文字前加上 - ， * ，+
